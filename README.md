## 📌 Tutorial

1 - Clonar o projeto do git

```bash
git clone https://github.com/Davi-Cesar/b2bit-davi-cesar.git
```

2 - instale as depêndencias na raiz do projeto

```bash
cd b2bit-davi-cesar

yarn
```

3 - Execute o projeto

```bash
yarn start
```
